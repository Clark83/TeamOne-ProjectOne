Fork Off!

Trying to find a place to eat?

Here's how to get started:

Step 1: One person in your party needs to initialize a room so that a ROOM KEY code can be shared.

Step 2: Share the room key code once a room has been initialized.

Step 3: Once all users are in the room, click the "Fork Off" button to start deciding!

Step 4: A confirmation message will appear as soon as there is a matched decision.

Step 5: Click the "More Info" button to get directions, reviews, and contact info to the restaurant.
